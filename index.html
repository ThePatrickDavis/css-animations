Keyframes

Keyframes are used to create a graduation change in styles over time. Keyframes give more control over intermediate steps of an animations than transitions. 

@keyframes fade-in {
    0% {
        opacity: 0;
    }
    100% {
        opacity: 1;
    }
}

Animation Properties

You can apply a @Keyframes to any object by using the animation css properties and shown below. 

.slide {
    left: -100%;
    position: relative;
    animation-name: slide-in;
    animation-duration: 1s;
    animation-delay: .5s;
    animation-iteration-count: 1;
    animation-timing-function: ease-in;
    animation-fill-mode: forwards;
    /* Or use Shorthand */
    animation: slide-in 1s ease-in .5s 1 forwards;
}

Vendor Prefixes

Gross... we have to type everything in triplicate, but that's CSS baby! 

@-moz-keyframes rotate { 100% { -moz-transform: rotate(360deg); } }
@-webkit-keyframes rotate { 100% { -webkit-transform: rotate(360deg); } }
@keyframes rotate { 100% { -webkit-transform: rotate(360deg); transform:rotate(360deg); } }

*I'd Create a SASS mixin if I was doing a real project.

Multiple Animations

You can combine multiple animations by separating them with commas. 

.slide5 {
    animation: rotate 4s linear 0s 1,
                spaz 4s linear 0s 1,
                color-change 4s linear 0s 1;
}
